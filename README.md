# Design-Patterns-in-Modern-Cpp
Documenting Code Implementations and Reflections from Learning Design Patterns

## 🛠 开发环境
* **操作系统**: Ubuntu 24.04 LTS (Linux)
* **编译器**: GCC 13.3.0 (支持 C++20/23)
* **构建工具**: CMake 3.16+
* **语言标准**: C++20

## 🚀 如何运行
1. 克隆仓库: `git clone https://github.com/你的用户名/Cpp20-Design-Patterns.git`
2. 进入目录: `cd Cpp20-Design-Patterns`
3. 创建构建目录: `mkdir build && cd build`
4. 编译: `cmake .. && cmake --build .`
